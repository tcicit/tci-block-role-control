= Switch blocks on and off per user role =

== Description ==
Simple plugin to switch blocks on and off based on user roles.

== Languages ==
English, German, German Switzerland

== Installation ==
1. download the plugin and upload it to the directory `wp-content/plugins`.
2. activate the plugin.

= How does the plugin work? =
The plugin allows administrators to enable or disable blocks based on user roles.

= Blöcke pro Benutzerrolle ein und ausschalten =

== Description ==
Einfaches Plugin um Blöcke auf basis von Benutzerrollen ein- und aus zu schalten.

== Sprachen ==
Englisch, Deutsch, Deutsch Schweiz

== Installation ==
1. Plugin herunterladen und in das Verzeichnis `wp-content/plugins` hochladen.
2. Plugin aktivieren.

= Wie funktioniert das Plugin? =
Das Plugin ermöglicht es Administratoren, Blöcke basierend auf Benutzerrollen ein- oder auszuschalten.
